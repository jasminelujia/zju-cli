# ZJU CLI Tools

zju wlan

zju vpn -c // config

zju vpn // connect

if ip 10.189.xxx.xxx => zjuwlan => set up routes
else => set up default routes

zju vpn --zjuwlan => is zjuwlan

zju vpn -d // disconnect
